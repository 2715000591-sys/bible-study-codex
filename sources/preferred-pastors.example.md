# 偏好牧师名单模板

复制一份到 `sources/preferred-pastors.local.md`，填入你自己常看的牧师。`*.local.md` 默认不会上传 GitHub。

## 格式

```md
- 牧师名字：
  - 常看教会/机构：
  - 常看网站或频道：
  - 备注：
```

## 示例

```md
- 牧师名字：示例名字
  - 常看教会/机构：示例教会
  - 常看网站或频道：https://example.com
  - 备注：只查这个官方频道，避免同名混淆
```

## 使用方式

你可以只给 Codex 牧师名字。更稳的做法是补充教会、网站或频道，减少同名误判。
