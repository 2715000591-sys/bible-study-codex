# Get笔记使用教学

第一版不做账号联动。Get笔记负责录音、转写、说话人识别；Codex 负责整理转写稿。

## 建议流程

1. 小组开始前打开 Get笔记录音。
2. 尽量让每个人靠近麦克风说话。
3. 结束后导出文字稿。
4. 如果 Get笔记能识别说话人，保留说话人标签。
5. 把文字稿粘贴给 Codex，并说：

```text
用 $bible-study-codex 总结这份Get笔记转写稿。请按成员观点、共同看见、分歧、后续问题整理。
```

## 注意

- 如果转写稿没有说话人标签，Codex 不应该猜谁是谁。
- 如果涉及隐私，不要把原文上传到公开 GitHub。
- 总结时可以隐去姓名，用“成员A、成员B”。
